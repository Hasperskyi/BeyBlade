BeyBlade JavaFX Game

![Build](https://github.com/YOUR_USERNAME/BeyBlade/actions/workflows/build.yml/badge.svg)

A small desktop game written in Java and JavaFX.
The project was created as a final assignment for a Java Advanced course.
It demonstrates a simple game interface, sound effects and basic game logic.


Features:
- JavaFX graphical interface
- Sound effects (JavaFX Media)
- Maven build system
- Portable desktop application
- Automatic build using GitHub Actions


Run from Source:
- Clone the repository:
git clone https://github.com/YOUR_USERNAME/BeyBlade.git
- Run the application:
mvn javafx


Download:
- Prebuilt versions can be downloaded from the Actions tab.
- Download the artifact: BeyBlade-windows.zip
- Extract and run: BeyBlade.exe


Technologies:
- Java
- JavaFX
- Maven
- JUnit
- GitHub Actions
